# 🧲 NMR Spin Simulation (Bloch Equation Visualization)

> Interactive visualization of nuclear spin dynamics, precession, and signal generation using Python and Pygame.

---

## 🌟 Highlights

* 3D-inspired visualization of magnetization vectors
* Simulation of Bloch equation dynamics
* Larmor precession under a static magnetic field
* Free Induction Decay (FID) signal generation
* FFT-based frequency spectrum visualization
* Models transmit (Tx) and receive (Rx) coil behavior

---

## ℹ️ Overview

This project simulates the microscopic behavior of hydrogen nuclei in the presence of a static magnetic field and RF excitation, inspired by principles of Nuclear Magnetic Resonance (NMR).

The goal is to provide an **intuitive, visual understanding** of:

* Spin alignment and precession
* Signal generation and decay (FID)
* Frequency-domain analysis via FFT

This simulation is part of a broader effort to design and build a **low-cost desktop NMR spectrometer**.
